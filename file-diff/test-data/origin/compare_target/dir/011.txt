我们很高兴地宣布，He3最新版本 (2023.1.16) 已经发布。这次更新带来了许多新功能和改进，使He3更加强大和易用。我们相信这次更新将满足我们用户的需求，并为他们带来更好的体验。
下面是此次更新的全部内容:

系统功能更新
支持通过 Scoop 安装
更为简洁的设置页界面
设置更新即点即生效
左侧菜单栏支持自定义分类
Mac 下支持 Command + , 快捷键打开设置
支持拖动工具，调整位置
工具支持一键置顶
工具支持修改所在分类
更多工具的支持输入框内容的自动填充 ，减轻粘贴操作
更多工具支持搜索框预览
精准的智能灯泡推荐
右键支持快捷复制选项
输入框自动聚焦
更多国家和地区的日期格式支持
应用内所有工具整体布局优化，统一间距
修复了 MacOS 下黑暗模式滚动条颜色适配问题
新增工具
Paseto 生成器
文本分析
JSON/YAML 转 PHP Array
UTF7 编码、解码
UTM 生成器
CSS 边框圆角生成器
CSV 类转换工具，目前支持 CSV 与 Markdown、HTML、JSON、SQL 的互转
请求发送器
公式计算器
图片比较：对比不同清晰度下的区别
工具更新
「Unicode 解码」 支持忽略非 Unicode 内容
「Paseto 解析器」升级，支持指定 SigningKey，并支持 JSON 和 List 不同展示结果
「JWT 解析器」 升级，将 Header 部分和 Payload 分别开来，更准确地展示结果
「JWT 生成器」 支持更多的加密算法，包括非对称加密算法
「JSON 转 YAML」两边编辑器都可以进行编辑实现实时互相转换
「JSON 和 YAML 转换工具类」支持错误提示和错误行高亮
「条形码生成器」以及「生成二维码类工具」支持导出成 SVG 文件
「加解密类工具」支持更多文件类型
「正则表达式」对匹配结果高亮显示，同时可以快速替换
「图片类工具」支持直接 Ctrl + V（Mac 下为 Command + V粘贴）快捷键进行粘贴
「X509 证书生成」采用异步生成形式，不会造成应用卡死
「随机时间生成器」、「时间戳转换」支持更多格式输出
「时间戳转换」支持选择不同时区
「我的 IP」支持刷新功能
